# Multi-Platform Matrix Build Demo

This repository demonstrates a **GitHub Actions matrix build** with artifact management.

- Matrix builds for Node.js versions: 14, 16, and 18
- Each build runs in parallel and uploads a unique artifact named:
  - `build-bcfd918-v14`
  - `build-bcfd918-v16`
  - `build-bcfd918-v18`
- Identifier step: `matrix-bcfd918`
